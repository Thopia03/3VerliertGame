## How to start

To start the Application, you need to have Python 3.14 installed. That is all that is needed. If you run into any problems, please send me a message.

If you have your code in your editor of choice, do the following:

1) move from the root to the 'code' directory
2) use the promt ```py -m src.starter``` to start the application!
